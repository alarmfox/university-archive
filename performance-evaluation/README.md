# Performance Evaluation

This directory contains materials for the Performance Evaluation course.

## Contents

- [notes/](./notes/): LaTeX notes for the course.
- [benchmark/](./benchmark/): Benchmark scripts and data.
- [capacity-test/](./capacity-test/): Capacity testing material.
- [design-of-experiment/](./design-of-experiment/): Design of Experiment (DoE) material.
- [fairness-index/](./fairness-index/): Fairness index analysis.
- [field-failure-data-analysis/](./field-failure-data-analysis/): Analysis of field failure data.
- [linear-regression/](./linear-regression/): Linear regression homework and analysis.
- [pca-and-clustering/](./pca-and-clustering/): PCA and Clustering exercises.
- [workload-characterization/](./workload-characterization/): Workload characterization material.

## Notes

The notes are written in LaTeX. To compile them, use `latexmk` or any LaTeX editor.
